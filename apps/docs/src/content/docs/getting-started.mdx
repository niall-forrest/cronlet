## 1. Install

```bash
npm install cronlet cronlet-cli
```

## 2. Initialize the project

```bash
npx cronlet init
```

`cronlet init` detects your project, creates a jobs directory, adds an example job, and writes `cronlet.config.*`.

## 3. Create a job

Create `src/jobs/weekly-digest.ts` (or `jobs/weekly-digest.ts` depending on your project):

```ts
import { schedule, weekly } from "cronlet";

export default schedule(weekly("fri", "09:00"), async () => {
  await sendWeeklyDigest();
});
```

## 4. Start local dev scheduler

```bash
npx cronlet dev
```

You will get:

- job discovery
- local scheduler execution
- dashboard at `http://localhost:3141`
- file watching and hot reload

## 5. Verify jobs are discoverable

```bash
npx cronlet list
```

Run one job manually:

```bash
npx cronlet run weekly-digest
```

Validate config and schedule declarations:

```bash
npx cronlet validate
```

## Optional: add runtime safety

```ts
import { schedule, every } from "cronlet";

export default schedule(
  every("1m"),
  {
    // default is "skip"
    concurrency: "skip",
    // only used with concurrency: "skip"
    catchup: true,
  },
  async () => {
    await doWork();
  }
);
```

## Timeout and retry duration format

Use integer + unit values:

- `100ms`
- `30s`
- `5m`
- `1h`
- `1d`

These units are supported for both `timeout` and `retry.initialDelay`.
