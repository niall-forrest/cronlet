---
title: Introduction
description: What cronlet is and when to use it.
---

`cronlet` is a TypeScript-first scheduling toolkit for Node.js apps.

It helps you:

- define schedules with typed builders (`every`, `daily`, `weekly`, `monthly`, `cron`)
- discover jobs from your codebase with stable IDs
- run jobs locally with a dev dashboard
- deploy scheduled routes for platforms like Vercel

Use `cronlet` when you want scheduling integrated directly into your existing app code without introducing a separate orchestration platform.
